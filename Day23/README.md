# üê¢ Turtle Crossing Game (Python)

A simple arcade-style game built using Python's `turtle` module.  
The player controls a turtle and must cross a busy road while avoiding moving cars.  
Each successful crossing increases the level and car speed.

---

## Features

- Increasing difficulty with levels
- Randomly generated cars
- Collision detection
- Live level tracking
- Clean object-oriented structure

---

## Concepts Used

- Python OOP (classes & objects)
- Turtle graphics
- Event listeners (keyboard input)
- Game loop logic
- Collision detection

---

## How to Play
- Press **SPACEBAR** to move the turtle forward
- Avoid hitting the cars
- Reach the top of the screen to level up
- Game ends on collision

---

## Project Structure
- Project23.py # Main game loop
- player.py # Player (turtle) logic
- cars.py # Car creation & movement
- scoreboard.py # Level display & game over

---

## How to Run
1. Make sure all files are in the same folder
2. Run the main file: Project23.py
